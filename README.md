# Universite Management System

University Management System (UNCHK) est une application web de gestion universitaire développée dans le cadre d'un projet académique.

L'objectif principal est de centraliser la gestion administrative, pédagogique et professionnelle des étudiants au sein d'un établissement universitaire.

L'application permet notamment la gestion :

des utilisateurs et rôles ;
des étudiants ;
des formations ;
des inscriptions ;
des stages ;
de l'insertion professionnelle ;
des partenaires ;
des réunions ;
des documents administratifs ;
des statistiques et tableaux de bord.
